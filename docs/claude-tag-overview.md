# Claude Tag 解説

> 本ドキュメントは、Anthropic が発表した「Claude Tag」のベータ提供開始に関するお知らせ（Slack スレッドで共有）を解説したものです。
> 元情報: [@claudeai の投稿 (X)](https://x.com/claudeai/status/2069468693017268244)

## TL;DR

- **Claude Tag** は **Claude Code の進化形**にあたる新機能。
- Slack 上で **Claude をメンション（タグ付け）してリクエスト**すると、タスクを自律的に処理して結果をスレッドに返してくれる。
- **2026年時点で、Claude Enterprise / Team プランの Slack でベータ版**として利用可能。

## Claude Tag とは

Claude Tag は Claude Code をさらに発展させたもので、以下を特徴とします。

- **より積極的（プロアクティブ）に動作する**: 受け身で指示を待つだけでなく、タスクの達成に向けて主体的に動く。
- **フルチームでの作業を前提に設計されている**: 個人の開発支援にとどまらず、チーム全体のワークフローへ組み込まれることを想定している。

## 動作の流れ

Slack スレッド内で Claude をタグ付けしてリクエストを送ると、Claude は次のように動きます。

1. **タスクを段階（ステップ）に分解する**
2. **アクセス可能なツールを使って各ステップを処理する**
3. **作成した成果物をスレッド内に返信する**

つまり、Slack のスレッドが「依頼から成果物の受け取りまで」を完結させる作業スペースになります。

## 利用条件

| 項目 | 内容 |
| --- | --- |
| 提供形態 | ベータ版 |
| 対象プラットフォーム | Slack |
| 対象プラン | Claude Enterprise / Claude Team |

> Team プランでも利用できるため、チームでの試用が可能です。
